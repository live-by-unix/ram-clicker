## Ram Clicker - A fun remake of Cookie Clicker
Ram Clicker is a cookie clicker - gambling (pls do not try in real life) ascending, buying things, MHZ and Dollars instead of cookies, and more!

This README covers tech stack, licensing, and has a quick preview

## Tech Stack
* React + Vite
* HTML
* CSS
* JS (JSX)
